Sample HTML Projects

A sandpit for playing around with the Internet
and testing web technologies.

All websites are hosted in Azure or Github Pages. Source code is available in GitHub and linked within each project.

JavaScript Examples

Modulo Multiplication Tables in JavaScript. Drawing geometric patterns on a canvas using client-side JavaScript. Demonstrates using event-emitters and listeners.
The Royal Game of Ur. Coding a board game user-interface and a PVE computer AI using client-side JavaScript (Under Construction).

Asp.Net Examples

MyNoddy MVC Supermarket Sweep. An online shopping game in a store with a depreciating stock count on the server updated in real-time on the client. Demonstrates an in-process server function that simulates a shopper-bot working as an AI adversary. Hosted as an Azure webapp and uses a fully responsive layout.
A WebForms Application. Old-school webforms example. To be completed (Under Construction).

Azure Cloud Examples

Modulo Multiplication Tables in JSON. Drawing geometric patterns using JSON objects, Azure [serverless] functions and Azure Cloud Services. Demonstrates use of a continuous asynchronous request-response loop using XmlHttpRequest (XHR) and promises.
Node.Js Web Server. To be completed (Under Construction).
